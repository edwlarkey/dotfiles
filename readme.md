# Requirements

* [docopt](http://docopt.org/) -- [docopt - github](https://github.com/docopt/docopt)


# Install


run bin/edw

    Usage:
        edw install <apps>...
        edw update
        edw upgrade
        edw dotfiles [--clone] (--home | --work)
        edw -h | --help
        edw --version
